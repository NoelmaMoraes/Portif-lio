Projeto: Portfólio Pessoal

Este projeto consiste em uma landing page desenvolvida com o objetivo de apresentar um portfólio pessoal na área de desenvolvimento web.

A página apresenta informações sobre o estudante, incluindo uma breve descrição pessoal, habilidades técnicas e comportamentais, além de uma lista de projetos desenvolvidos durante os estudos na área de programação.

O projeto foi desenvolvido utilizando HTML para a estrutura da página, CSS para a estilização visual e JavaScript para pequenas interações e funcionalidades da página.

O objetivo principal deste projeto é reunir em um único espaço digital informações profissionais e projetos desenvolvidos durante o curso, servindo como uma forma inicial de apresentação profissional na área de tecnologia.

A página contém seções de apresentação, habilidades, projetos e contato, permitindo que visitantes conheçam o perfil do estudante e suas produções.

Este projeto poderá ser atualizado futuramente com novos projetos e novas habilidades adquiridas ao longo da trajetória na área de desenvolvimento web.
